# ファイル整理

## 手順
1. **スコープ確認**: 対象ディレクトリ、問題、除外対象、積極度
2. **現状分析**: ファイル数・種別・サイズ分布
3. **パターン選択**: 種類別/目的別/日付別
4. **重複検出**: `find -exec md5 {} | sort | uniq -d`（削除前に必ず確認）
5. **計画提示→承認→実行**
6. **サマリ**: 整理数・削減容量・新構造

## 命名規則
`2024-10-17-meeting-notes.md` — 日付入り、スペース避ける、重複番号除去

## 分類パターン
| パターン | 例 |
|---------|-----|
| 種類別 | Documents/, Images/, Code/ |
| 目的別 | Work/Active/, Personal/Archive/ |
| 日付別 | 2024/Q1/, 2024/Q2/ |

## ルール
- 削除前に必ず確認、全移動をログ記録
- 6ヶ月以上未使用→アーカイブ候補
