# 開発ブランチ完了処理

## フロー
テスト検証（失敗なら停止）→ ベースブランチ特定 → 4選択肢提示 → 実行 → クリーンアップ

## 選択肢
| # | アクション | マージ | プッシュ | Worktree | ブランチ削除 |
|---|----------|--------|---------|---------|------------|
| 1 | ローカルマージ | o | - | 削除 | -d |
| 2 | PR作成 | - | -u origin | 保持 | - |
| 3 | そのまま保持 | - | - | 保持 | - |
| 4 | 破棄 | - | - | 削除 | -D（確認必須） |

## 禁止
テスト失敗のまま進行、確認なし削除、無断force push
