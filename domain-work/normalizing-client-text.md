# S1: NormalizeClientText

表記ゆれ（前株/後株、半角カナ、全角英数、空白）を吸収し、学習データのキーマッチング率を向上。

## 正規化ルール
1. **NFKC正規化**: 全角英数→半角、半角カナ→全角、㈱→(株)
2. **法人格除去**: 株式会社/有限会社/合同会社/(株)/(有) を削除
3. **空白正規化**: 連続空白→半角スペース1つ、前後Trim
4. **英字小文字化**: `.toLowerCase()`

## テストケース（重要）
| Input | Output |
|---|---|
| `（株）ABC` | `abc` |
| `ｼｬｶｲﾎｹﾝﾘｮｳ` | `シャカイホケンリョウ` |
| `ABC 12月分` | `abc 12月分`（月度は維持） |
| `三井　住友` | `三井 住友` |

## 注意
- `12月`や`No.1`等の識別子まで削除しない（過剰な削除は禁止）
- 空白は「全削除」ではなく「連続→1つ」（区切りを消すと`ABC12`と`ABC1`が混ざる）
