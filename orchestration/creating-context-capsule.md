# Context Capsule作成

コンテキストリセット/セッション終了前に作業状態を保存。

## 手順

### 1. sessions.md を追記
```markdown
## YYYY-MM-DD セッションN
### 実施内容
- （箇条書き）
### 変更ファイル
### 変更規模
### 未完了・申し送り
```

### 2. decisions.md を更新（方針決定があった場合のみ）

## ルール
- sessions.md は追記のみ（過去の記録を編集しない）
- 簡潔に書く（長いログは次回のトークンを食う）
