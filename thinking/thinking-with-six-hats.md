# Six Hats Thinking

## 6つのハット
| 色 | 役割 | 問うこと |
|----|------|----------|
| 青 | メタ/進行 | 今何を考えるべきか。議題設定と結論統合 |
| 白 | 事実/データ | 確定事実、数字、不確定情報、情報の欠落 |
| 赤 | 感情/直感 | 好き嫌い、不安、ワクワク、違和感 |
| 黒 | リスク/弱点 | 最悪ケース、隠れたコスト、失うもの |
| 黄 | メリット/価値 | 最善ケース、得られる経験、広がる選択肢 |
| 緑 | アイデア/代替 | 未検討案、組み合わせ、前提の転換 |

## 推奨順序
```
青(開始) → 白 → 赤 → 黒 → 黄 → 緑 → 青(結論)
```
軽い判断なら白→黒→黄→結論でも可。

## ルール
1. **1度に1色だけ。混線禁止。**
2. 青ハットが流れをコントロール
3. 各ハットは最低1つの具体的アウトプット（「特にない」禁止）
4. 全ハット発言してから結論
5. 白ハットでは意見を言わない（事実と意見を分離）
